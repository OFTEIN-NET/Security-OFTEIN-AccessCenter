1. Tunnel Mode (SSH Protocol)

Currently we only support Windows and Linux Operating Systems.

Windows users need to download Putty SSH client and update registry.
Linux users need to download the script and execute with options.

Windows Machine : Putty SSH Client | Windows Registry for Putty
Linux Machine       : Linux Shell Script

2. Open Virtual Desktop Portal (HTTP Protocol)

It will required java web plugins, please ensure your java plugin updated.
Some browser maybe have some security or plugins issues, please check with your browser supports

Open Virtual Desktop Page

3. Virtual Machine Remote Desktop (RDP Protocol)

Currently we only support Windows and Linux Operating Systems.

Windows users will utilized default windows remote desktop client.
Linux users will "rdesktop" as remote desktop client.

Windows Machine : Open Remote Desktop
Linux Machine       : Linux Shell Script
